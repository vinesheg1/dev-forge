# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-02-05

### Added
- Initial release of dev-forge
- Biome integration for JS/TS/JSON linting and formatting
- Knip integration for dead code detection
- Lefthook integration for Git hooks
- Commitlint integration for conventional commits
- Stylelint integration for CSS/SCSS linting
- npm-package-json-lint integration for package.json validation
- `forge init` command to scaffold configurations
- `forge check` command to run all quality checks
- `forge fix` command to auto-fix issues
- Comprehensive documentation and examples
